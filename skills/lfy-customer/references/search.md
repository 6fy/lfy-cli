# search — 搜索客户

按关键字搜索客户列表，支持模糊匹配。

## 命令格式

```bash
lfy-cli customer search '{"keywords": "<keywords>"}'
```

## 参数说明

| 参数      | 类型   | 必填 | 说明                    |
| --------- | ------ | ---- | ----------------------- |
| `keywords` | string | 是   | 搜索关键字，支持模糊匹配 |

## 返回格式

```json
[
  {
    "customer_id": 81969602858,
    "customer_name": "四川未来科技",
    "gtm_id": 81929351422
  },
  {
    "customer_id": 81969602859,
    "customer_name": "北京科技有限公司",
    "gtm_id": 81929351423
  }
]
```

## 返回字段说明

| 字段           | 类型    | 说明                      |
| -------------- | ------- | ------------------------- |
| `customer_id`  | integer | 客户唯一 ID               |
| `customer_name` | string  | 客户名称                  |
| `gtm_id`       | integer | GTM 分类 ID（业务线标识） |

## 筛选策略

`search` 返回结果后，在本地对客户名称进行筛选匹配：

| 场景                                    | 策略                           |
| --------------------------------------- | ------------------------------ |
| 精确匹配（customer_name 完全一致）      | 直接使用，向用户展示结果       |
| 模糊匹配（customer_name 包含关键词）     | 直接使用，向用户展示结果       |
| 模糊匹配，找到多个结果                   | 展示候选列表，请用户选择       |
| 无匹配结果                              | 告知用户未找到对应客户         |

## 错误处理

- `keywords` 为空时返回错误或不完整结果
- 若 `errcode` 不为 `0`，展示错误信息并终止流程
- 若返回格式异常，告知用户并建议稍后重试
